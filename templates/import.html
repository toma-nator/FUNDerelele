Upload a TD Direct Investing or CIBC Investor's Edge export. Accepts CSV files (both brokers) or PDF account statements (TD only). Duplicate transactions are automatically skipped.

How to export from TD Direct Investing (CSV — 2 years)
  1. Log in → Accounts → Account Activity
  2. Set your date range, then click Download
  3. Choose CSV format and save the file
  4. Upload the file above
TD account statements (PDF — full history)
  1. Log in → Accounts → Statements & Documents
  2. Download monthly or quarterly PDF statements
  3. Upload each PDF above (select TD Direct Investing or Auto-detect)
How to export from CIBC Investor's Edge
  1. Log in → My Portfolio → Transaction History
  2. Set your date range and click Export
  3. Choose CSV and save
  4. Upload the file above
